🌿 Sistema IoT Ambiental

EcoNodo

Sistema IoT de monitoreo ambiental

EcoNodo captura datos del entorno mediante sensores conectados a un ESP32, los envía a la nube y los presenta en una plataforma web clara y accesible para facilitar el monitoreo ambiental.

  • Mide temperatura, humedad, presión, PM2.5, PM10, VOC y calidad del aire.
  • Envía lecturas a Supabase mediante HTTPS.
  • Muestra dashboard, historial, alertas y configuración desde la web.
ESP32 + Sensores → Supabase → Dashboard Web
